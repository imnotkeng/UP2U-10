## Project Overview
This project is to create a complete and functional Flood Fill puzzle game.
The goal is not to over-engineer, but to build a clear, playable prototype with core mechanics:
- random grid generation
- color flooding
- limited moves
- territory expansion
- win/loss conditions

This project follows a vibe coding approach:

---
## Core Goal
Create a Flood Fill game that:
- generates a grid of random neon colors
- allows player to control the top-left tile
- expands territory when adjacent colors match
- limits the number of moves to add challenge
- provides visual feedback (animations) on color change
- supports restart or instant replay
- is fully playable
The output must be a complete working game with clear commented code.
